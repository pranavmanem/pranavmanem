MicroCar

This micro-car was designed as a project for the class Human Factors at Georgia Tech. The idea was to create an ergonomic design that allowed the four humans. Based on anthropometric data, a criteria was determined that the car would fit from the 5th to 95th percentile human.

Shoes for Marsabit

As part of a team helping the non-profit, Partners for Care, design a shoe making process that could be used by the Rindelli tribe in Kenya to fabricate footwear. This was done through a unique process modeled after Japanese Waraji rope sandals. The rope used for the soles was made from french-braided corn husks, which was then wrapped in fabric to improve comfort and aesthetics. Learn the Process

Creative Decisions
and Design

In the class, Creative Decisions and Design, my team was tasked with building a machine to compete in a competition utilizing machining programming skills. It's subsystems consisted of a telescoping arm, a ramp, a piston powered arm, and a limit switch system for arm control.

Batmobile Tumbler 3D CAD Model

In the Engineering Graphics and Visualization course at Georgia Tech, we were tasked with using Autodesk Inventor to create a 3D model. We decided to model and assemble over fourty parts using the skills we had obtained.

Shifting Omni-Directional
Gearbox Module

In my junior high school, I was part of an FRC Robotics team that did not have a student run CAD team. Using online resources, I taught myself Autodesk Inventor. To practice my skills, I created a shifting omni-directional module that could help a robot go up to 12 ft/s.

Modular Chassis

Another project that I worked on as part of my FRC team was designing a modular robotics chassis. The purpose of this was to create a system with all the components necessary to test subsystems and save time during the build season.
